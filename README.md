# yairet-torrecilla
1. Uso de Cookies

Las cookies son pequeños archivos que un sitio web guarda en el navegador del usuario para almacenar información, como preferencias, idioma o datos de acceso. Esto permite ofrecer una experiencia más personalizada.

2. Creación de Sesiones

Las sesiones permiten identificar a un usuario después de iniciar sesión. Gracias a ellas, el usuario puede navegar por diferentes páginas sin tener que volver a introducir sus credenciales en cada una.

3. Eliminación de Sesiones

La eliminación de una sesión ocurre cuando el usuario cierra sesión (logout) o cuando la sesión expira por inactividad. Esto elimina los datos temporales de autenticación y ayuda a proteger la seguridad de la cuenta.
